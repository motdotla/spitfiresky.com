Beginner's guide to Unicorn

Ahoy. Thar' be unicorns. Here I show you how to get started using unicorn with nginx.

First things first

Unicorn is a nice replacement for mongrel and thin, and a good alternative to phusion passenger. GitHub uses unicorn. I am assuming your server setup is similar to my server setup.

Installation

Install the unicorn gem.

Configure your nginx virtual host. It should look something like this. Take care to make sure the paths match correctly with where you are storing your application on the server.

In your merb or rails application create a config/unicorn.rb file. (If you are deploying a rails app you would replace instances of RACK_ENV with RAILS_ENV.)

Add a restart unicorn task to your deploy file.

Then deploy your app. With a little luck, you will be live.
