Saturday 5:00pm
Goldmine is setup to work with Google apps on Gina’s computer (I was able to access it remotely). The process is a bit involved because we have to wrap it in SSL using stunnel (Google apps requires SSL authentication but Goldmine 7 does not support SSL authentication.)

Here are the steps:

Phase 1 (stunnel configuration).

The instructions are below, but you can also watch the phase 1 screencast.

  1. Download stunnel to the desktop
  2. Install the program by double clicking on it
  3. Once installed go to Start → Programs → Stunnel → Edit stunnel.conf
  4. Modify as follows
    1. Uncomment the “client = yes” line (remove the ; )
    2. modify the [pop3s] section as follows:
      1. accept = 995
      2. connect = pop.gmail.com:995
    3. modify the [sstmp] section as follows:
      1. accept = 465
      2. connect = smtp.gmail.com:465
  5. Save that file and close it
  6. Click Start → Programs → Stunnel → Service Install
  7. and Click Start → Programs → Stunnel → Run Stunnel
